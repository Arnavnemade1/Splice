Get started.

Three lines. Then it runs entirely on your machine.

git clone https://github.com/Arnavnemade1/Splice.git
cd Splice && npm install && npm run build
claude mcp add splice -- node dist/cli.js start
01 Node 20+ The only prerequisite.
02 Claude Code, Desktop & Cursor splice init prints the snippet for each.
03 No cloud, no telemetry Everything stays local. State encrypted at rest.

That's the whole install.
