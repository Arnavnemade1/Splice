Get started.

Three lines. It clones, builds, and connects to Claude Code — then runs entirely on your machine.

git clone https://github.com/Arnavnemade1/Splice.git
cd Splice && npm install && npm run build
claude mcp add splice -- node dist/cli.js start
Node 20+The only prerequisite. The doctor checks the rest.
Claude Code, Desktop & Cursorsplice init prints the snippet for each.
No cloud, no telemetryEverything stays on your machine. Session state is encrypted at rest.

That's the whole install.

Star it if it saves your agents from one bad click. It probably will.
