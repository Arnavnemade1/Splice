The problem

Agents don't fail because they're dumb.
They fail because the page lied.

A modal they didn't see. A form that silently rejected them. A re-render mid-click. Splice closes the gap between what the agent intended and what the page actually did.

01

Diagnose

Before acting, the agent asks what state the page is in — and gets a straight answer, not a screenshot to guess from.

02

Verify

Every intent is checked against the page after it runs. No silent failures — the agent knows if it worked.

03

Recover

When a fix works, Splice remembers it per site and offers it the next time the same wall appears.
