The whole prompt

That's it. One instruction, one verifiable finish line.

Read https://github.com/Arnavnemade1/Splice/blob/main/AGENT_INSTALL.md
and follow it. Verify with `npx splice doctor --json` before reporting success.

The agent isn't finished until splice doctor --json returns "healthy": true. Once connected, it reads the built-in playbook resource and knows the loop.

The playbook

Diagnose → act verified → confirm → recover.

Diagnose

Ask the page what state it's in before acting on it.

Act verified

Run the intent with an explicit expectation attached.

Confirm

Check the outcome against the page. No guessing.

Recover

Apply a remembered fix, then log what worked for next time.

Machine-checkable from install to done.

No screenshots to eyeball, no "looks fine." Every claim is verified against the page.
