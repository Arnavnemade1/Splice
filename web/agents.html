The contract

Three commands, one finish line.

Requires Node 20+ — the doctor checks everything else.
01 · Clone Fetch the source. git clone https://github.com/Arnavnemade1/Splice.git
02 · Build Succeeds when dist/cli.js exists. cd Splice && npm install && npm run build
03 · Connect For Claude Desktop or Cursor, run init instead — it prints each snippet. claude mcp add splice -- node dist/cli.js start
Done when This returns "healthy": true. Not before. node dist/cli.js doctor --json
On failure The doctor names the failed check and its remedy. Apply it, re-run the same command.

Machine-checkable, start to finish.

No screenshots to eyeball. No "looks fine."
