Get started

Setup

From zero to verified
in three steps.

Every step is a command with a machine-checkable result. You're done when the doctor says so — not before.

1

Install

Clone, build, and health-check. The doctor tells you exactly what to fix if anything is off.

git clone https://github.com/Arnavnemade1/Splice.git
cd Splice && npm install && npm run build
node dist/cli.js doctor
2

Connect your agent

One line for Claude Code. splice init prints snippets for Claude Desktop and Cursor.

claude mcp add splice -- node dist/cli.js start
3

Let it work

The agent diagnoses each page before acting, runs intents that are verified after the fact, and reads only what changed.

compile_verified_action { intent: 'click "Place order"',
  expect: [{ kind: "url_contains", value: "/thanks" }] }
The loop

Once connected, every task runs the same reliable cycle.

01

Diagnose

Read the page's real state before touching it.

02

Act, verified

Run the intent and confirm the outcome against the page.

03

Recover

Hit a wall? Apply a remembered fix and log what worked.

Two minutes to a healthier agent.

No accounts, no cloud, no telemetry. It runs entirely on your machine.
